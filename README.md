# Arch Technologies - Cyber Security Internship

## [cite_start]Task 2: Key Logging [cite: 6]
[cite_start]This project involves simulating a basic keylogger in a safe and controlled environment to understand keystroke logging mechanisms. [cite: 7]

### Features
* [cite_start]Captures user keystrokes locally. [cite: 8]
* [cite_start]Logs data into a file (`keylog.txt`). [cite: 8]
* [cite_start]Analyzes potential risks associated with such attacks. [cite: 8]

### Requirements
* Python 3.x
* `pynput` library

### How to Run
1. Install dependencies: `pip install pynput`
2. Run the script: `python keylogger.py`
3. Press 'Esc' to stop logging and save the file.
